Riven AI/Get Started/Python SDK

Python SDK

Async-first Python SDK with full type hints and Pydantic models.

The Python SDK provides an idiomatic Python interface to Riven AI with full async/await support, type hints, and Pydantic models for request/response validation. Works seamlessly with FastAPI, Django, and other Python frameworks.

Features

Async/Await Native

Built from the ground up with asyncio for high-performance applications.

Full Type Hints

Complete type annotations for IDE autocompletion and static analysis.

Pydantic Models

Request and response validation using Pydantic v2 models.

Sync Support

Synchronous client available for scripts and simple applications.

Installation

bash
pip install riven-sdk
# or with poetry
poetry add riven-sdk
# or with uv
uv add riven-sdk

Quick Start

Here is a minimal example to get started with the Python SDK:

main.py
1from riven import Riven AI
2import asyncio
3
4# Initialize the client
5riven = Riven AI(
6    base_url="http://localhost:8080",
7    api_key="your-virtual-key",  # Optional
8)
9
10async def main():
11    # Make a chat completion request
12    response = await riven.chat.completions.create(
13        provider="openai",
14        model="gpt-4o-mini",
15        messages=[
16            {"role": "user", "content": "Hello, how are you?"}
17        ],
18        max_tokens=100,
19        temperature=0.7,
20    )
21
22    print(response.choices[0].message.content)
23
24asyncio.run(main())

Streaming Responses

Use async generators for real-time streaming responses:

streaming.py
1from riven import Riven AI
2import asyncio
3
4riven = Riven AI(base_url="http://localhost:8080")
5
6async def stream_chat():
7    stream = await riven.chat.completions.create(
8        provider="openai",
9        model="gpt-4o-mini",
10        messages=[
11            {"role": "user", "content": "Write a short poem about coding"}
12        ],
13        max_tokens=500,
14        stream=True,
15    )
16
17    async for chunk in stream:
18        content = chunk.choices[0].delta.content
19        if content:
20            print(content, end="", flush=True)
21    print()
22
23asyncio.run(stream_chat())

Using Fallbacks

Configure automatic failover between providers:

fallbacks.py
1from riven import Riven AI
2import asyncio
3import json
4
5riven = Riven AI(base_url="http://localhost:8080")
6
7async def chat_with_fallbacks():
8    response = await riven.chat.completions.create(
9        provider="openai",
10        model="gpt-4o-mini",
11        messages=[
12            {"role": "user", "content": "Explain quantum computing"}
13        ],
14        max_tokens=1000,
15        extra_headers={
16            # Fallback chain: OpenAI -> Anthropic -> Bedrock
17            "x-model-fallback": json.dumps([
18                {"provider": "anthropic", "model": "claude-3-5-sonnet-20241022"},
19                {"provider": "bedrock", "model": "anthropic.claude-3-sonnet-20240229-v1:0"}
20            ])
21        }
22    )
23
24    # Check which provider actually served the request
25    print(f"Response from {response.provider}:")
26    print(response.choices[0].message.content)
27
28asyncio.run(chat_with_fallbacks())

OpenAI SDK Compatibility

You can also use the official OpenAI Python SDK by pointing it to your Riven AI gateway:

openai_compat.py
1from openai import OpenAI
2
3# Point OpenAI SDK to Riven AI
4client = OpenAI(
5    base_url="http://localhost:8080/openai/v1",
6    api_key="your-virtual-key",  # Or any string if not using virtual keys
7)
8
9completion = client.chat.completions.create(
10    model="openai/gpt-4o-mini",  # provider/model format
11    messages=[
12        {"role": "user", "content": "Hello!"}
13    ],
14)
15
16print(completion.choices[0].message.content)
Using the OpenAI SDK is the recommended approach for existing codebases. Simply change the base_url to your Riven AI gateway and you're ready to go!

FastAPI Integration

Example of using Riven AI with FastAPI:

app.py
1from fastapi import FastAPI
2from fastapi.responses import StreamingResponse
3from riven import Riven AI
4import json
5
6app = FastAPI()
7riven = Riven AI(base_url="http://localhost:8080")
8
9@app.post("/chat")
10async def chat(message: str):
11    response = await riven.chat.completions.create(
12        provider="openai",
13        model="gpt-4o-mini",
14        messages=[{"role": "user", "content": message}],
15    )
16    return {"response": response.choices[0].message.content}
17
18@app.post("/chat/stream")
19async def chat_stream(message: str):
20    async def generate():
21        stream = await riven.chat.completions.create(
22            provider="openai",
23            model="gpt-4o-mini",
24            messages=[{"role": "user", "content": message}],
25            stream=True,
26        )
27        async for chunk in stream:
28            content = chunk.choices[0].delta.content
29            if content:
30                yield f"data: {json.dumps({'content': content})}\n\n"
31        yield "data: [DONE]\n\n"
32
33    return StreamingResponse(generate(), media_type="text/event-stream")
