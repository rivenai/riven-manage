Google GenAI SDK Integration

Use the Google GenAI SDK with Riven AI through the native Gemini generateContent API format.

Riven AI exposes a Gemini-compatible endpoint at /genai/v1beta/models.

Python

python
import google.generativeai as genai

genai.configure(
    api_key="any",
    client_options={"api_endpoint": "http://localhost:8080/genai"},
)

model = genai.GenerativeModel("gemini-2.0-flash")
response = model.generate_content("What is the meaning of life?")
print(response.text)

Streaming

python
response = model.generate_content("Tell me a story", stream=True)
for chunk in response:
    print(chunk.text, end="")

Multimodal

python
import PIL.Image
image = PIL.Image.open("image.jpg")

response = model.generate_content(["What is in this image?", image])
print(response.text)

TypeScript

typescript
import { GoogleGenerativeAI } from "@google/generative-ai"

const genAI = new GoogleGenerativeAI("any")
// Override the base URL by setting the environment variable or clientOptions
const model = genAI.getGenerativeModel({ model: "gemini-2.0-flash" })
const result = await model.generateContent("Hello!")
