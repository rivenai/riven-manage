Riven AI/Features/Batch Processing

Batch Processing

Process thousands of requests efficiently with up to 50% cost savings.

Batch processing allows you to submit large volumes of requests that are processed within a 24-hour window at significantly reduced costs. Perfect for data enrichment, content generation, and analytics workloads.

Overview

The Batch API is designed for high-volume, latency-tolerant workloads. Submit a file containing multiple requests, and Riven AI will process them efficiently, returning results in a single output file.

50% Cost Savings

Batch requests are priced at half the standard rate.

24-Hour SLA

Results guaranteed within 24 hours, often much faster.

100K+ Requests

Process up to 100,000 requests per batch file.

Creating a Batch

Step 1: Prepare Your Input File

Create a JSONL file with one request per line:

batch_input.jsonl
1{"custom_id": "request-1", "method": "POST", "url": "/v1/chat/completions", "body": {"model": "openai/gpt-4o-mini", "messages": [{"role": "user", "content": "Summarize this article: ..."}]}}
2{"custom_id": "request-2", "method": "POST", "url": "/v1/chat/completions", "body": {"model": "openai/gpt-4o-mini", "messages": [{"role": "user", "content": "Translate to Spanish: ..."}]}}
3{"custom_id": "request-3", "method": "POST", "url": "/v1/chat/completions", "body": {"model": "openai/gpt-4o-mini", "messages": [{"role": "user", "content": "Extract entities from: ..."}]}}

Step 2: Upload the File

bash
curl -X POST http://localhost:8080/v1/files \
  -H "Authorization: Bearer $API_KEY" \
  -F "purpose=batch" \
  -F "file=@batch_input.jsonl"

# Response
{
  "id": "file-abc123",
  "object": "file",
  "bytes": 1024567,
  "created_at": 1731672000,
  "filename": "batch_input.jsonl",
  "purpose": "batch"
}

Step 3: Create the Batch

bash
curl -X POST http://localhost:8080/v1/batches \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $API_KEY" \
  -d '{
    "input_file_id": "file-abc123",
    "endpoint": "/v1/chat/completions",
    "completion_window": "24h"
  }'

# Response
{
  "id": "batch_abc123",
  "object": "batch",
  "endpoint": "/v1/chat/completions",
  "input_file_id": "file-abc123",
  "completion_window": "24h",
  "status": "validating",
  "created_at": 1731672000,
  "request_counts": {
    "total": 1000,
    "completed": 0,
    "failed": 0
  }
}

Batch Statuses

StatusDescription
validatingInput file is being validated
in_progressRequests are being processed
finalizingResults are being compiled
completedAll requests processed successfully
failedBatch failed (see error file)
expiredBatch exceeded completion window
cancelledBatch was cancelled

Retrieving Results

Download the output file once the batch completes:

bash
curl http://localhost:8080/v1/files/file-output123/content \
  -H "Authorization: Bearer $API_KEY" \
  -o batch_output.jsonl

Output File Format

batch_output.jsonl
1{"id": "response-1", "custom_id": "request-1", "response": {"status_code": 200, "body": {"choices": [{"message": {"content": "Summary: ..."}}], "usage": {"prompt_tokens": 100, "completion_tokens": 50}}}}
2{"id": "response-2", "custom_id": "request-2", "response": {"status_code": 200, "body": {"choices": [{"message": {"content": "Traducción: ..."}}], "usage": {"prompt_tokens": 80, "completion_tokens": 90}}}}
3{"id": "response-3", "custom_id": "request-3", "error": {"code": "rate_limit_exceeded", "message": "Rate limit exceeded after retries"}}

Python Example

Complete example using the Python SDK:

batch_example.py
1import json
2import time
3from openai import OpenAI
4
5client = OpenAI(
6    base_url="http://localhost:8080/openai/v1",
7    api_key="your-api-key"
8)
9
10# Prepare batch input
11requests = [
12    {"custom_id": f"req-{i}", "method": "POST", "url": "/v1/chat/completions",
13     "body": {"model": "gpt-4o-mini", "messages": [{"role": "user", "content": f"Question {i}"}]}}
14    for i in range(1000)
15]
16
17# Write to JSONL file
18with open("batch_input.jsonl", "w") as f:
19    for req in requests:
20        f.write(json.dumps(req) + "\n")
21
22# Upload file
23with open("batch_input.jsonl", "rb") as f:
24    input_file = client.files.create(file=f, purpose="batch")
25
26# Create batch
27batch = client.batches.create(
28    input_file_id=input_file.id,
29    endpoint="/v1/chat/completions",
30    completion_window="24h"
31)
32
33print(f"Batch created: {batch.id}")
34
35# Poll for completion
36while batch.status not in ["completed", "failed", "expired", "cancelled"]:
37    time.sleep(60)
38    batch = client.batches.retrieve(batch.id)
39    print(f"Status: {batch.status}, Progress: {batch.request_counts.completed}/{batch.request_counts.total}")
40
41# Download results
42if batch.status == "completed":
43    output = client.files.content(batch.output_file_id)
44    with open("batch_output.jsonl", "wb") as f:
45        f.write(output.read())
46    print("Results downloaded to batch_output.jsonl")
