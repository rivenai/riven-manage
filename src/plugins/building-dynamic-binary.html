Dynamic Binary Plugins

Build plugins as standalone executables in any language using a simple stdin/stdout JSON protocol. The gateway spawns the binary as a subprocess and communicates via newline-delimited JSON.

Binary plugins run as subprocesses — safe to write in Python, Node.js, Ruby, or any language. For maximum performance, use Go or WASM plugins instead.

Protocol overview

The gateway writes one JSON object per line to the plugin's stdin, and the plugin responds with one JSON object per line on stdout:

json
// Input (stdin) — pre_llm_hook
{"hook": "pre_llm_hook", "request": {"model": "gpt-4o", "messages": [...]}, "metadata": {}}

// Output (stdout) — continue to LLM
{"continue": true, "request": {"model": "gpt-4o", "messages": [...]}}

// Output (stdout) — short-circuit with a mock response
{"continue": false, "response": {"choices": [{"message": {"content": "cached result"}}]}}

Supported hooks

HookWhenCan short-circuit
pre_llm_hookBefore the provider request is sentYes
post_llm_hookAfter the provider response is receivedNo
post_llm_hook_asyncAfter response is sent to caller (non-blocking)No

Python example

python
#!/usr/bin/env python3
import sys
import json

for line in sys.stdin:
    payload = json.loads(line.strip())
    hook = payload["hook"]

    if hook == "pre_llm_hook":
        req = payload["request"]
        # Inject a system prompt
        req["messages"].insert(0, {
            "role": "system",
            "content": "You are a concise assistant. Respond in under 50 words."
        })
        print(json.dumps({"continue": True, "request": req}), flush=True)

    elif hook == "post_llm_hook":
        resp = payload.get("response", {})
        content = resp.get("choices", [{}])[0].get("message", {}).get("content", "")
        # Transform response — uppercase all content
        if content:
            resp["choices"][0]["message"]["content"] = content.upper()
        print(json.dumps({"response": resp}), flush=True)

    elif hook == "post_llm_hook_async":
        # Fire-and-forget: log usage
        usage = payload.get("response", {}).get("usage", {})
        sys.stderr.write(f"tokens_used={usage.get('total_tokens', 0)}\n")
        print(json.dumps({}), flush=True)

Register the plugin

json
{
  "plugins": [
    {
      "name": "my-python-plugin",
      "path": "/plugins/my_plugin.py",
      "type": "binary",
      "config": {
        "timeout_ms": 5000
      }
    }
  ]
}

Node.js example

javascript
#!/usr/bin/env node
const readline = require("readline")

const rl = readline.createInterface({ input: process.stdin })

rl.on("line", (line) => {
  const payload = JSON.parse(line)

  if (payload.hook === "pre_llm_hook") {
    const req = payload.request
    // Add metadata header
    req.metadata = { ...req.metadata, processed_by: "node-plugin" }
    process.stdout.write(JSON.stringify({ continue: true, request: req }) + "\n")
  } else {
    process.stdout.write(JSON.stringify({}) + "\n")
  }
})
