Vector Store

Vector database integration for semantic caching and similarity search.

Supported Backends

BackendTypeBest For
qdrantDedicated vector DBProduction, high performance
pgvectorPostgreSQL extensionExisting Postgres infrastructure
redisRedis StackExisting Redis infrastructure
memoryIn-memoryDevelopment, testing

Configuration

config.json
{
  "vector_store": {
    "type": "qdrant",
    "url": "http://localhost:6333",
    "collection": "riven_cache",
    "embedding_model": "text-embedding-3-small",
    "embedding_dimensions": 1536,
    "similarity_threshold": 0.95
  }
}

Vector Store Interface

pkg/vector/store.go
type VectorStore interface {
    // Store adds a vector with associated data
    Store(ctx context.Context, entry *CacheEntry) error
    
    // Search finds similar vectors
    Search(ctx context.Context, query []float32, limit int) ([]SearchResult, error)
    
    // Delete removes a vector by ID
    Delete(ctx context.Context, id string) error
    
    // Clear removes all vectors
    Clear(ctx context.Context) error
}

type CacheEntry struct {
    ID        string
    Vector    []float32
    Prompt    string
    Response  string
    Model     string
    CreatedAt time.Time
    TTL       time.Duration
}

type SearchResult struct {
    Entry      *CacheEntry
    Similarity float32
}

Embedding Generation

Riven AI generates embeddings using configurable models:

go
func (c *SemanticCache) generateEmbedding(ctx context.Context, text string) ([]float32, error) {
    resp, err := c.embeddingClient.CreateEmbedding(ctx, &EmbeddingRequest{
        Model: c.config.EmbeddingModel,
        Input: text,
    })
    if err != nil {
        return nil, err
    }
    return resp.Embedding, nil
}

func (c *SemanticCache) Lookup(ctx context.Context, prompt string) (*CachedResponse, error) {
    // Generate embedding for the prompt
    embedding, err := c.generateEmbedding(ctx, prompt)
    if err != nil {
        return nil, err
    }
    
    // Search for similar prompts
    results, err := c.store.Search(ctx, embedding, 1)
    if err != nil {
        return nil, err
    }
    
    if len(results) > 0 && results[0].Similarity >= c.config.Threshold {
        return &CachedResponse{
            Response:   results[0].Entry.Response,
            Similarity: results[0].Similarity,
        }, nil
    }
    
    return nil, nil // Cache miss
}

Embedding Costs

Embedding generation incurs API costs. Consider using a local embedding model for high-volume caching scenarios.
