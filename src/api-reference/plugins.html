Plugins API

Inspect and manage loaded plugins at runtime via the management API.

List Plugins

GET /api/plugins

bash
curl http://localhost:8080/api/plugins \
  -H "Authorization: Bearer admin-token"
json
{
  "plugins": [
    {
      "name": "logging",
      "enabled": true,
      "version": "1.4.0",
      "hooks": ["PreLLMHook", "PostLLMHook"],
      "stats": { "pre_hook_avg_ms": 0.12, "post_hook_avg_ms": 2.4 }
    },
    {
      "name": "semantic-cache",
      "enabled": true,
      "version": "1.2.0",
      "hooks": ["PreLLMHook", "PostLLMHook"],
      "stats": { "cache_hit_rate": 0.34, "pre_hook_avg_ms": 8.7 }
    }
  ]
}

Get Plugin

GET /api/plugins/:name

bash
curl http://localhost:8080/api/plugins/logging \
  -H "Authorization: Bearer admin-token"

Enable / Disable Plugin

bash
# Disable a plugin
curl -X PATCH http://localhost:8080/api/plugins/semantic-cache \
  -H "Authorization: Bearer admin-token" \
  -H "Content-Type: application/json" \
  -d '{ "enabled": false }'

# Re-enable
curl -X PATCH http://localhost:8080/api/plugins/semantic-cache \
  -H "Authorization: Bearer admin-token" \
  -H "Content-Type: application/json" \
  -d '{ "enabled": true }'

Reload Plugin

POST /api/plugins/:name/reload — Hot-reloads the plugin without dropping active requests (enterprise only).

bash
curl -X POST http://localhost:8080/api/plugins/my-custom-plugin/reload \
  -H "Authorization: Bearer admin-token"
json
{ "status": "reloaded", "plugin": "my-custom-plugin", "duration_ms": 14 }

Plugin Config

PUT /api/plugins/:name/config — Update plugin configuration at runtime.

bash
curl -X PUT http://localhost:8080/api/plugins/semantic-cache/config \
  -H "Authorization: Bearer admin-token" \
  -H "Content-Type: application/json" \
  -d '{ "similarity_threshold": 0.95, "ttl_seconds": 7200 }'
