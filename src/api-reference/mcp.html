MCP API

REST and WebSocket endpoints for the Riven AI MCP gateway.

WebSocket Endpoint

Agents connect to the MCP gateway using the standard MCP WebSocket protocol:

text
ws://localhost:8080/mcp
wss://your-riven-domain.com/mcp  (TLS)
javascript
const ws = new WebSocket("ws://localhost:8080/mcp", {
  headers: { Authorization: "Bearer your-virtual-key" },
});

ws.on("open", () => {
  // Send MCP initialize request
  ws.send(JSON.stringify({
    jsonrpc: "2.0",
    id: 1,
    method: "initialize",
    params: {
      protocolVersion: "2024-11-05",
      capabilities: { tools: {} },
      clientInfo: { name: "my-agent", version: "1.0.0" },
    },
  }));
});

REST: List Tools

GET /api/mcp/tools

bash
curl http://localhost:8080/api/mcp/tools \
  -H "Authorization: Bearer your-virtual-key"
json
{
  "tools": [
    {
      "name": "read_file",
      "description": "Read the contents of a file",
      "inputSchema": {
        "type": "object",
        "properties": { "path": { "type": "string" } },
        "required": ["path"]
      }
    }
  ]
}

REST: Call Tool

POST /api/mcp/tools/call

bash
curl -X POST http://localhost:8080/api/mcp/tools/call \
  -H "Authorization: Bearer your-virtual-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "read_file",
    "arguments": { "path": "/etc/hostname" }
  }'
json
{
  "content": [{ "type": "text", "text": "my-server
" }],
  "isError": false
}

REST: List Servers

GET /api/mcp/servers

bash
curl http://localhost:8080/api/mcp/servers \
  -H "Authorization: Bearer admin-token"
json
{
  "servers": [
    {
      "name": "filesystem",
      "status": "connected",
      "tool_count": 8,
      "url": "ws://mcp-filesystem.internal:3000"
    }
  ]
}
