Runway

Runway provides state-of-the-art AI video generation models including Gen-3 Alpha Turbo for text-to-video and image-to-video workflows.

Configuration

config.json
{
  "providers": [
    {
      "key": "runway",
      "api_keys": [
        { "value": "your-runway-api-key" }
      ]
    }
  ]
}

Supported models

ModelTypeMax Duration
gen3a_turboText-to-video / Image-to-video10s
gen3aText-to-video / Image-to-video10s

Generate a video

curl
curl http://localhost:8080/v1/videos/generations \
  -H "Content-Type: application/json" \
  -H "x-bf-provider: runway" \
  -d '{
    "model": "gen3a_turbo",
    "prompt": "A serene mountain lake at sunrise with mist rising from the water",
    "duration": 10,
    "ratio": "1280:768",
    "seed": 42
  }'

Image-to-video

curl
curl http://localhost:8080/v1/videos/generations \
  -H "Content-Type: application/json" \
  -H "x-bf-provider: runway" \
  -d '{
    "model": "gen3a_turbo",
    "prompt": "Gentle waves lapping on shore",
    "image_url": "https://example.com/beach.jpg",
    "duration": 5
  }'
Video generation is asynchronous. Poll GET /v1/videos/generations/{task_id} until status is SUCCEEDED, then download from the returned output URL.
