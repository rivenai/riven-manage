Quick start

Terminal
# Run with Docker
docker run -p 8080:8080 \
  -e OPENAI_API_KEY=sk-... \
  ghcr.io/rivenai/gateway:latest

# Or use as a drop-in OpenAI replacement
curl http://localhost:8080/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer your-virtual-key" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello"}]
  }'

Open source features

All core features are open source and production-ready.

Drop-in Replacement

Replace existing AI SDK connections by changing just the base URL. Keep your code, gain fallbacks and governance.

Retries & Fallbacks

Seamless failover between providers and models. When your primary provider fails, Riven AI switches to backups automatically.

Load Balancing

Intelligent API key distribution with weighted load balancing, model-specific filtering, and automatic failover.

Virtual Keys

The primary governance entity. Control access permissions, budgets, rate limits, and routing per consumer.

Routing Rules

CEL-based routing rules to direct requests to specific models, providers, and keys with weighted strategies.

Budget & Rate Limits

Hierarchical cost control with budgets and rate limits at virtual key, team, and customer levels.

MCP Tool Filtering

Control which MCP tools are available per virtual key with strict allow-lists for security.

Semantic Caching

Intelligent response caching based on semantic similarity. Reduce costs and latency for similar queries.

Built-in Observability

Monitor every AI request in real-time. Track performance, debug issues, and analyze usage patterns.

Prometheus Metrics

Native Prometheus metrics via scraping or Push Gateway for monitoring and alerting.

OpenTelemetry

OTLP integration for distributed tracing with Grafana, New Relic, Honeycomb, and more.

Custom Plugins

Extensible middleware architecture. Build Go or WASM plugins for custom logic at any step.

Mocker Plugin

Mock AI provider responses for testing, development, and simulation without real API calls.

Async Jobs

Fire-and-forget async inference jobs with webhook callbacks for long-running completions.

Batch Processing

OpenAI-compatible batch API for high-throughput, cost-efficient processing of large request volumes.
